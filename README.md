# Система прогнозирования риска отчисления студентов

## Описание
Этот проект анализирует данные об успеваемости студентов и прогнозирует вероятность их отчисления. Используются методы машинного обучения для выявления студентов из группы риска.

## Входные данные
Файл `data/input.csv` должен содержать столбцы:
- `student_id` — ID студента
- `name` — имя студента
- `attendance_rate` — посещаемость (0–100)
- `avg_grade` — средний балл (0–100)
- `missed_assignments` — количество пропущенных заданий
- `login_frequency` — частота входа в LMS (входов в неделю)

## Выходные данные
Результат сохраняется в `data/output.csv`:
- `name` — имя студента
- `dropout_risk` — вероятность отчисления (0–1)
